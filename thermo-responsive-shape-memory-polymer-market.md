<p><strong>Market Segmentation</strong></p>
<p><strong>The Thermo-responsive Shape Memory Polymer Market Analysis by types is segmented into:</strong></p>
<p><ul><li>Type I</li><li>Type II</li></ul></p>
<p><p>Thermo-responsive Shape Memory Polymers can be classified into Type I and Type II markets based on their thermomechanical properties. Type I polymers exhibit a one-way shape memory effect, where they can return to their original shape upon heating. Whereas, Type II polymers show a two-way shape memory effect, allowing them to return to both their original and a secondary programmed shape. Both types of polymers find application in various fields such as biomedical devices, aerospace, textiles, and actuators.</p></p>
